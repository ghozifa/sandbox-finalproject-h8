# sandbox-finalproject-h8

# Step to use project: 

1. Go to Server Dir, run:
### `npx sequelize-cli db:create`
### `npx sequelize-cli db:migrate`
### `nodemon app.js`

2. Go to client-calendar dir, run:
### `npm start`
